# How to Download Retro64 as an App

## 📱 Mobile (iOS & Android)

### iPhone/iPad (iOS)
1. Open **Safari** browser
2. Go to: `https://boberrealyt.github.io/Retro64/`
3. Tap the **Share button** (arrow up icon)
4. Select **"Add to Home Screen"**
5. Name it **"Retro64"**
6. Tap **"Add"**

✅ The app now appears on your home screen like a native app!

### Android Phone/Tablet
1. Open **Google Chrome** (or any Chromium-based browser)
2. Go to: `https://boberrealyt.github.io/Retro64/`
3. Tap the **Menu button** (3 dots)
4. Select **"Install app"** or **"Add to Home Screen"**
5. Confirm the installation

✅ The app now appears in your app drawer and home screen!

## 🎮 Using the App

### First Launch
- Library is empty - add your first ROM
- Click **"📚 Library"** → **"➕ Add ROM"**
- Select your N64 ROM file

### Game Controls
- **D-Pad** - Left side navigation
- **Left Stick** - Analog movement
- **A/B/Z** - Action buttons (right side)
- **L/R** - Shoulder buttons
- **START** - Pause game
- **C-Stick** - Camera control

### Save Progress
1. Click **"💾 Save/Load"** during gameplay
2. Choose a save slot (1-10)
3. Your progress is automatically saved

### Reset Progress
1. Go to **"📚 Library"**
2. Find your game
3. Click **"🔄 Reset"**
4. Confirm deletion

### Apply Shaders
1. Click **"🎨 Shaders"** during gameplay
2. Choose an effect (CRT, Scan Lines, etc.)

## ❓ FAQ

**Q: Will my saves be lost if I uninstall the app?**
A: Yes, uninstalling will delete saved progress. Back them up first!

**Q: Can I play offline?**
A: Yes! After adding ROMs, the app works completely offline.

**Q: What if I have a PC and Mac?**
A: Install on both! Each device stores its own library and saves.

**Q: How much storage do I need?**
A: Depends on ROM size. N64 games range from 4-64 MB each.

**Q: Can I update the app?**
A: Updates happen automatically when you next launch it.
