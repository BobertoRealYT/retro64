# Retro64 - N64 Emulator

A fully-featured Nintendo 64 emulator built for modern web browsers with complete mobile support, save states, shaders, and more.

![Retro64](https://img.shields.io/badge/Retro64-N64%20Emulator-FF6B6B?style=for-the-badge)

## ✨ Features

- **Complete N64 Controller Support** - All buttons and analog sticks mapped for authentic gameplay
- **Save States** - Up to 10 save slots per game with timestamps
- **Shader Effects** - Multiple visual filters (CRT, Scan Lines, Vignette, Blur, etc.)
- **iOS Support** - Install as a native-like app on iPhone/iPad
- **Android Support** - Full PWA support with home screen installation
- **Responsive Controls** - Touch-optimized controls that work across all device sizes
- **ROM Library Management** - Organize and manage your game collection
- **Session Persistence** - All progress automatically saved between sessions
- **Offline Support** - Play downloaded games without internet connection
- **ROM Hacks Support** - Play modified versions like Zelda B3313 and other hacks
- **Fullscreen Mode** - Immersive gaming experience
- **Modern UI** - Beautiful dark theme with smooth animations

## 🚀 Quick Start

### 1. Install on Your Device

**iOS (iPhone/iPad):**
1. Open Safari
2. Visit your Retro64 URL
3. Tap the Share button
4. Select "Add to Home Screen"
5. Name it "Retro64"
6. Tap Add

**Android:**
1. Open Chrome
2. Visit your Retro64 URL
3. Tap the menu (3 dots)
4. Select "Install app" or "Add to Home Screen"

### 2. Add ROMs

1. Tap the "📚 Library" button
2. Select "➕ Add ROM"
3. Choose your N64 ROM file (.z64, .n64, .v64, .zip)
4. The game will appear in your library

### 3. Start Playing

1. Tap a game to load it
2. Use the on-screen controls to play
3. Tap "💾 Save/Load" to manage save states
4. Tap "🎨 Shaders" to apply visual effects

## 🎮 Controls

### Standard N64 Controller Layout

**Left Side:**
- D-Pad (↑↓←→) for navigation
- Left Stick for analog movement

**Right Side:**
- A Button (primary action)
- B Button (secondary action)
- Z Button (targeting/alternative)
- L & R Shoulder Buttons
- START Button to pause

**C-Stick:**
- C-Up, C-Down, C-Left, C-Right for camera and special moves

## 🎯 Save States

Save your progress at any point:
1. Tap "💾 Save/Load" button
2. Choose a save slot (1-10)
3. Your game state is instantly saved

Load a previous save:
1. Tap "💾 Save/Load"
2. Select the save state to restore
3. Continue from where you left off

### Reset Progress

To delete all save states for a game:
1. Open the Library
2. Find your game
3. Tap the "🔄 Reset" button
4. Confirm when prompted

## 🎨 Shader Effects

Apply visual filters to enhance your gaming experience:

- **None** - Original graphics
- **CRT** - Classic cathode ray tube monitor effect
- **Scan Lines** - Retro TV scan line effect
- **Retro** - Vintage arcade appearance
- **Blur** - Soft blur filter
- **Vignette** - Darkened edges effect

Access shaders by tapping "🎨 Shaders" button during gameplay.

## 🐛 Troubleshooting

### Game won't load
- Check ROM file format is supported
- Try a different ROM version
- Clear browser cache and reload

### Controls not responding
- Tap the screen to ensure it's focused
- Check that buttons aren't hidden
- Try toggling controls off/on

### Low performance
- Reduce visual effects/shaders
- Close other browser tabs
- Restart the app

### Save states not working
- Ensure you have storage space available
- Check browser storage settings
- Try deleting old save states
